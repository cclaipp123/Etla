## 🎯 彩蛋功能简介

这是一个专门记录大写转换工具中隐藏彩蛋的文档。这些彩蛋没有任何实际作用，纯粹是为了让枯燥的财务计算变得有趣一点。

## 🤔 为什么要有彩蛋？

1. **防止程序员无聊**：写财务代码太枯燥了，需要一点调剂
2. **测试用户好奇心**：看看有没有人真的会输入这些奇怪数字
3. **增加代码趣味性**：谁说财务软件不能有趣？
4. **纯粹的无聊**：就是觉得好玩

## 🎨 彩蛋设计理念

- **隐蔽性**：只有特定数字才能触发
- **趣味性**：结合网络文化梗
- **无害性**：不影响正常功能
- **短暂性**：3秒后自动消失，不打扰用户
